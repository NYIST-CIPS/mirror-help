=====================
Arch Linux 源使用帮助
=====================

地址
====

https://mirrors.ustc.edu.cn/archlinux/

说明
====

Arch Linux 软件源

收录架构
========

i686, x86_64

使用说明
========

编辑 :file:`/etc/pacman.d/mirrorlist` ，在文件的最顶端添加

::

    Server = https://mirrors.ustc.edu.cn/archlinux/$repo/os/$arch

国内用户，推荐另外使用 Arch Linux CN 的源，请参考 :doc:`archlinuxcn`
